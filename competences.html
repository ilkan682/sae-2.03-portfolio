Compétence 1 — Administrer les systèmes et les réseaux

C1
AC11.03
Configurer le réseau local

Configuration d'un adressage cohérent en mode pont pour la communication inter-machines sur le LAN. La VM Debian a obtenu une IP routable permettant au poste Windows de la joindre directement.

AC11.04
Maîtriser les systèmes d'exploitation

Gestion système complète sous Linux Debian : modification des fichiers de configuration réseau, gestion des politiques des démons système (MariaDB), édition de /etc/hosts.

Compétence 2 — Connecter les entreprises et les usagers

C2
AC12.05
Communiquer avec un tiers en situation professionnelle

Planification et déploiement collaboratif de comptes d'accès individualisés pour les personne du groupe (rafael, hussain), avec attribution des droits SQL.

💻

Compétence 3 — Développer des applications réseaux

C3
AC13.03
Traduire un algorithme en un programme

Écriture de scripts Python pour le traitement des formulaires web, la validation des données et les calculs dynamiques de moyennes par étudiant et par matière.

AC13.05
Choisir les mécanismes de gestion de données adaptés

Liaison via connecteur SQL (mysqlclient) d'un framework web objet (Django) à une base de données MariaDB externe hébergée sur infrastructure virtuelle .
