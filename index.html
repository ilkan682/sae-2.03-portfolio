Contexte du Projet

SAÉ 2.03

Dans le cadre du deuxième semestre de notre BUT Réseaux et Télécommunications, nous avons développé une Application Web de Gestion de Notes intitulée gestion-notes.

L'objectif principal de cette SAÉ était de maîtriser le framework Django et de comprendre l'architecture client-serveur globale.

  • L'application Django s'exécute sur une machine hôte ou une machine virtuelle dédiée.
  • La base de données MariaDB est hébergée et isolée sur un serveur virtuel distinct sous Debian Linux via VirtualBox.

L'enjeu technique résidait dans la mise en place de la configuration réseau, de l'adressage et des autorisations d'accès SQL afin de lier ces deux environnements de manière sécurisée et transparente.

👥

Équipe Projet

Collaboratif

Ce projet a été mené en équipe de trois étudiants :

  • Ilkan GUNEY (BDD & Élèves) — Modélisation et sécurisation SQL, développement du système CRUD (Ajout/Modification d'étudiants) et configuration de la liaison à la base de données.
  • Hussain (Matières & Serveur) — Gestion de la connexion BDD, affichage des matières, développement du CRUD matières et configuration réseau.
  • Rafael (CSV & Bulletins) — Conception des interfaces HTML/CSS, lecture et traitement des fichiers CSV pour l'importation automatisée des notes dans la base de données.
