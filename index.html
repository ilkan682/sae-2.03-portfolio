Contexte du Projet

SAÉ 2.03

Pour le deuxième semestre du BUT , nous avons développé une Application Web de Gestion de Notes qui s'appelle gestion-notes.

L'objectif de cette SAÉ était de maîtriser Django et de comprendre comment fonctionne l'architecture client-serveur.

  • L'application Django s'exécute sur une machine virtuelle .
  • La base de données MariaDB est hébergée sur un serveur virtuel qui est sous Debian Linux via VirtualBox.

Les difficultés technique était de faire les configuration réseau pour que la basses de donné soit accessible a distance avec des utilisateurs .

👥

Équipe Projet

Ce projet a été fait avec une équipe de 3 :

  • Ilkan GUNEY (BDD & Élèves) — Modélisation et sécurisation SQL, développement du système CRUD (Ajout/Modification d'étudiants) et configuration de la liaison à la base de données.
  • Hussain (Matières & Serveur) — Gestion de la connexion BDD, affichage des matières, développement du CRUD matières et configuration réseau.
  • Rafael (CSV & Bulletins) — Conception des interfaces HTML/CSS, lecture et traitement des fichiers CSV pour l'importation automatisée des notes dans la base de données.
