📝 Decentralized Web Application Architecture

The primary target of this academic project was to engineer and deploy a decentralized grade management web platform called "gestion-notes" using the Django framework. In order to match real-world enterprise infrastructure designs, we decoupled the service layers into two different technical environments: the web app backend runs on our local Windows hosts, while the MariaDB relational database is securely isolated inside a remote Debian virtual machine managed through VirtualBox.

My duties during this deployment focused on network architecture design, relational database security management, and backend scripting. I successfully set up a bridged network layer to grant the virtual machine its own valid IP address on the local area network (LAN), revised MariaDB's server configuration templates to drop local-only restrictions, and created secure individual accounts for my team members. Furthermore, I engineered Python backend logic to process user workflows, compute academic averages, and serve real-time database results onto dynamic web interfaces. This comprehensive assignment enhanced my engineering skills in Linux system deployment, data structure mapping, and network-level troubleshooting.
